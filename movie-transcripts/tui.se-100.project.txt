Create a file called cat-says.
touch cat-says

Make it executable.
chmod +x cat-says

Then you can use the following code to draw the cat image.

SHOW: vim cat-says

Execute the program.
./cat-says

If you make the program cat-says <argument>, it will print the cat image with the argument.
